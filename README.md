# llm-compliance

面向**离线场景的轻量级多模态提示注入检测**算法框架研究。

> **项目定位**：在 LLM 入口处构建一个**轻量、可打包、跨平台、零网络依赖**的"前置过滤"组件。区别于业界主流"在线大模型防御"（Lakera Guard、PromptGuard 等云端方案），本项目聚焦于**离线 + 轻量**象限，研究目标设备为**普通 PC 与主流边缘设备**。
>
> **研究产出**：一套 backbone-agnostic 的检测算法框架（而非单一模型），同一套训练与评测代码既能用于本项目选用的轻量 backbone，也能平滑迁移到 7B / 13B 规模的视觉-语言模型继续研究。

## 核心差异化（与业界"在线大模型防御"路线对比）

| 维度 | 业界主流 | 本项目 |
|---|---|---|
| 部署形态 | 云端 API | 端侧 / 边缘独立部署 |
| 模型规模 | 数百 MB ~ 数十 GB | < 500MB（默认） |
| 网络依赖 | 必需 | **无** |
| 数据流向 | 需上传 prompt 至云端 | 数据完全在本地 |
| 隐私属性 | 数据出境风险 | 数据不出端 |
| 适用场景 | 通用 SaaS 防护 | 隐私敏感、边缘、低成本合规 |

## 重要声明

本项目**仅用于安全研究、合规审计与防御研究**：
- **不发布为对外可调用的在线服务或 API**；
- **不构建主动攻击工具或越狱教程**；
- **不替代**模型训练阶段的安全对齐，仅作前置过滤 / 审计用途；
- 研究目标设备为普通 PC 与主流边缘设备，**不覆盖 MCU 级**极端嵌入式。
